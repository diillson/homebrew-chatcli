# Homebrew Tap for ChatCLI

Official Homebrew tap for [ChatCLI](https://github.com/diillson/chatcli) — a command-line interface for interacting with multiple LLM providers.

## Installation

```bash
brew tap diillson/chatcli
brew install chatcli
```

## Upgrade

```bash
brew upgrade chatcli
```

## Uninstall

```bash
brew uninstall chatcli
brew untap diillson/chatcli
```

## About ChatCLI

ChatCLI is a powerful CLI tool that supports multiple LLM providers including OpenAI, Anthropic Claude, Google Gemini, AWS Bedrock, DeepSeek, GitHub Copilot, and more.

Features:
- Interactive chat with streaming responses
- Agent mode with autonomous task execution
- Coder mode for code generation and editing
- Context management for large codebases
- Plugin system for extensibility
- Multi-language support (English, Portuguese)
- Session management and history

For more information, visit [chatcli.ai](https://chatcli.ai).
